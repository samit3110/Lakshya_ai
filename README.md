# 🌐 Jarvis Interface - Live Assistant Terminal

An ultra-futuristic, high-fidelity AI Voice and Text Assistant Interface modeled after the Jarvis terminal concept. This application integrates advanced **Groq API acceleration** for real-time speech-to-text (Whisper) and intelligence (Llama 3), alongside an **unlimited, high-quality local text-to-speech engine (Edge-TTS)**. 

Designed specifically for institutional guides, parents, and visitor orientation terminals, the AI is completely bounded to a custom infrastructure knowledge base to guarantee 100% factual accuracy with **zero hallucinations**.

---

## 🔥 Features & UI Matrix

* **3D Particle Hologram Dynamic Sphere**: A responsive HTML5 canvas animation running mathematical spatial transformations that adapts visually to the terminal's state (`STANDBY`, `LISTENING`, `PROCESSING`, `SPEAKING`, `ERROR`).
* **Dual Interaction Modality**: Fluid support for both hands-free voice calls with real-time automatic silence detection and an integrated dark-terminal keyboard chat lane.
* **Intelligent Overlap Termination**: Next-generation state management instantly cuts off active audio playback the exact moment a new query is submitted, preventing voice stacking.
* **Adaptive Vocal Token Stripping**: Displays extensive technical infrastructure data and facility blueprints clearly on screen, while reading out an elegant, minimal verbal summary to save audio bandwidth.
* **Strict Fact Bounding Framework**: Locked to a zero-temperature strict contextual mapping schema to ensure information is pulled exclusively from verified local facts.

---

## 🛠️ Tech Stack & Dependencies

* **Frontend Engine**: Pure Vanilla HTML5, CSS3 Terminal Matrix Styling, Web Audio API, Canvas 3D Projection Logic.
* **Backend Framework**: Python 3.11+, Flask.
* **AI Orchestration**: Groq SDK (`whisper-large-v3` for speech transcription, `llama-3.1-8b-instant` optimized at `temperature=0.0`).
* **Speech Generation**: Asynchronous `edge-tts` (Microsoft Neural Audio Architecture - Voice profile: `en-US-ChristopherNeural`).
